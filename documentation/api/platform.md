## PLATFORM
PENDINT TO IMPLEMENT

* `GET /platforms/`
> Get list of platforms
* `GET /platforms/{id}/`
> Get platform details
* `POST /platforms/`
> Create platform
* `PUT /platforms/`
> Update platform
* `DELETE /platforms/`
> Delete platform
